Step 1: Locate the image that you will like to use as a favicon and make sure the dimensions are a perfect square i.e. 40 pixels x 40 pixels

Step 2: Browse to the file location on your computer and select the image

Step 3: Press the “Generate favicon” button to create a favicon

Using the generator tool, a favicon can be created from text.

Use the options to pick your preferred colors; pick font color as well as background color.

Please choose transparent if you desire a transparent background

Step 1: Create a folder in your project directory named Images (if you haven’t done this already)

Step 2: Save the desired favicon image into this folder

Step 3: Add the HTML code generated to your index.html file right below the <"title"> element

Step 4: Replace the favicon_image_location witht the file path of your favicon image

Step 5: Save and reload

The favicon must be placed in the <"head"> element of your project. If this has happend, the favicon should display properly. Also, since different platforms use varied sizes; ensure the sizing correspond appropriately to the target platform
